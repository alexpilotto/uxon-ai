## Article
Use **Layout & Appearance** settings to control the overall visual feel of your landing page without editing each section one by one.

## Open Layout & Appearance settings

1. Open your landing page in Builder.
2. Select **Settings** in the left sidebar.
3. Open **Layout & Appearance**.

## Set your page width

- Use **Page width** to make your page feel tighter or wider on desktop.
- A narrower width can improve readability for long-form copy.
- A wider width can work better for visual-heavy sections.

## Adjust corner radius across key elements

- **Cards corner radius**: controls card-style blocks.
- **Sections corner radius**: controls section container shape.
- **Form inputs corner radius**: controls input field style.

## Recommended workflow

1. Set layout width first.
2. Choose corner style direction (sharp, medium, rounded).
3. Check Desktop and Mobile preview before saving.

> Keep shape style consistent across buttons, cards, and inputs for a cleaner, more trustworthy look.

## Troubleshooting

**The page still looks inconsistent after updates.**

- Some sections may use their own local overrides. Open that section's settings and review local styling.

**Desktop looks good but mobile feels crowded.**

- Reduce visual density and recheck spacing in Mobile preview.
