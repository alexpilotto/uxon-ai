## Article
Style copy/paste helps you standardize design quickly, especially when you are polishing multiple sections before launch.

## Copy style from a source element

1. Select the source element in the canvas.
2. Open **Element settings** (gear icon).
3. Click **Copy style** in the settings header.

## Paste style to another element

1. Select the target element.
2. Open **Element settings**.
3. Click **Paste style**.
4. Review the result in Desktop and Mobile preview.

## When this is most useful

- Standardizing multiple CTA buttons.
- Applying consistent text treatment across similar blocks.
- Speeding up final QA before publishing.

> Use style copy/paste for consistency, but still sanity-check spacing and readability per section context.

## Troubleshooting

**Paste style did not change everything.**

- Some style properties may differ between element types. Recheck the target element settings after paste.

**Design looks inconsistent after pasting.**

- Re-apply style from the correct source element and validate in mobile preview.
