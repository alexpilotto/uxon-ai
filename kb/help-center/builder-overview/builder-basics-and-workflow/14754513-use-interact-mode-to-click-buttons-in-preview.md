## Article
If page elements are not clickable in Builder preview, you are likely in edit mode. Switch to **Interact mode** to test real behavior.

## Enable Interact mode

1. In Builder top bar, click **Interact with preview** (cursor icon).
2. When active, test your page by clicking links, buttons, and forms.

## What to test in Interact mode

- Primary and secondary CTA links
- Navigation anchors and section jumps
- Form interactions and submit behavior
- Phone and external links

## Test across viewports

1. Run checks in **Desktop** preview.
2. Repeat in **Mobile** preview.
3. Save before final review.

> Interact mode is best used as a quick pre-publish QA pass, especially for paid-traffic landing pages.

## Troubleshooting

**I still cannot click elements.**

- Confirm Interact mode is enabled, then retest.

**Clicks work on desktop but not mobile.**

- Review mobile layout and spacing in Mobile preview, then test again in Interact mode.
