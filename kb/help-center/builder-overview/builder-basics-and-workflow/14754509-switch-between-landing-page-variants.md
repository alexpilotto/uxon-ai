## Article
Use the variant switcher in Builder to edit the correct version of your landing page without leaving your workflow.

## Where to find the variant switcher

1. Open a landing page in Builder.
2. In the top bar, open the variant dropdown next to the landing page name.
3. Select the variant you want to edit.

## What happens when you switch variants

- Builder opens the selected variant's layout and content.
- Any edits you make apply to that variant only.
- Preview and final page URL update to the selected variant context.

## Recommended variant workflow

1. Name variants clearly (for example by angle, audience, or offer).
2. Switch variants before editing to avoid changes in the wrong version.
3. Save and preview each variant separately.

> Treat each variant like its own campaign page. Keep messaging, structure, and CTA decisions intentional per variant.

## Troubleshooting

**I cannot find the variant I need.**

- Check your landing page variant list and naming to confirm it exists.

**I edited the wrong variant by mistake.**

- Switch to the correct variant immediately, then apply changes there and recheck before publishing.
