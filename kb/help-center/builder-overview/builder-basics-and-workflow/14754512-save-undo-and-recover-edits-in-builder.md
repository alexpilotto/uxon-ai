## Article
Builder gives you multiple safety controls so you can edit quickly without losing important landing page work.

## How saving works in Builder

- Use the **Save** button in the top-right when you finish a change set.
- Use keyboard shortcut **Cmd/Ctrl + S** to save faster.
- Builder can auto-save while you keep working.

## Use undo and redo during iteration

- **Undo**: step back from the latest change.
- **Redo**: reapply a reverted change.

This is useful during rapid copy/design testing.

## Exit safely when changes are pending

- If you try to leave with unsaved edits, Builder prompts you to save or discard.
- Choose **Save** when you want to keep the latest version.

## Recommended save routine

1. Make a focused set of edits.
2. Save.
3. Preview the page and key actions.
4. Repeat.

> Frequent saves reduce rework and make collaboration safer when multiple teammates review the same page.

## Troubleshooting

**I clicked back and almost lost work.**

- Use the unsaved changes prompt to save before leaving.

**My latest tweak is not visible in preview.**

- Save first, then reopen preview and test again.
