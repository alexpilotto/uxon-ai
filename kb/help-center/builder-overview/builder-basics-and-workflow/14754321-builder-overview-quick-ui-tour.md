## Article
The UXON AI Builder is designed to help marketers launch and improve landing pages quickly. This guide gives you a simple tour so you can start editing with confidence.

## What you see in the top bar

- **Landing page + variant selector**: switch between variants without leaving Builder.
- **Quick setup**: apply core styling defaults fast.
- **Interact with preview**: click through buttons, forms, and links as a visitor would.
- **Desktop / Mobile / Responsive preview**: check layout across device views.
- **Undo / Redo**: reverse or re-apply recent changes.
- **Open landing page** (eye icon): open the current page URL in a new tab.
- **Save**: manually save changes any time.

## Left toolbar and side panels

The vertical toolbar includes:

- **Layout**: manage sections and page structure.
- **Settings**: edit page-level settings like brand assets, typography, colours, buttons, SEO, and more.
- **Code**: open custom code tools.
- **Help**: open support directly from Builder.

## Canvas and section controls

- Click a section in the canvas to select it.
- Use the floating controls to open settings, duplicate, hide, or delete sections.
- Use the blue **+** controls above or below a selected section to insert new sections exactly where you want them.

## A simple first-session workflow

1. Open the right variant.
2. Set baseline style in **Quick setup** or **Settings**.
3. Build your section flow in **Layout**.
4. Check **Mobile preview** before saving.
5. Select **Save** and open the page with the eye icon for a final pass.

> Best practice: make one focused change at a time, save, and preview. This keeps iteration fast and reduces rework.

## Troubleshooting

**I cannot click page elements in preview.**

- Enable **Interact with preview** in the top bar.

**I do not see the section list.**

- Open **Layout** from the left toolbar.

**I changed something but cannot see it live yet.**

- Select **Save**, then reopen the page using the eye icon.
