## Article
Use this workflow when you want to build a landing page from scratch (or rebuild a variant) in a clean, structured way.

## Before you start

- Open the correct landing page and variant in Builder.
- Decide your primary goal (for example leads, demo bookings, or purchases).
- Prepare your offer copy, proof points, and CTA wording.

## 1) Create your page structure

1. Go to **Layout**.
2. Select **Add** to open **Browse Sections**.
3. Add your core sections in order (for example hero, benefits, proof, offer, FAQ, final CTA).

## 2) Customize each section

- Select a section in the canvas or section list.
- Open settings and update text, media, buttons, and layout options.
- Rename sections in the Layout list so your structure stays easy to manage.

## 3) Apply page-level settings

1. Open **Settings** from the left toolbar.
2. Set brand assets (logo, favicon, sharing image), typography, colours, and button styles.
3. Complete SEO fields like **Page title** and **Meta description**.

## 4) Check mobile before final save

- Switch to **Mobile preview**.
- Review spacing, readability, and CTA placement.
- Adjust section or element settings where needed.

## 5) Save and preview

1. Select **Save**.
2. Use the eye icon to **Open landing page** in a new tab.
3. Run one final click-through of key actions (buttons, forms, links).

> Start with structure first, then styling. Teams move faster when the section flow is locked before visual polishing.

## Troubleshooting

**I added a section but it appears in the wrong spot.**

- Use drag-and-drop in the Layout list, or insert sections with the + controls above/below a selected section.

**My page feels too long.**

- Hide low-priority sections first, then delete only if you are sure they are unnecessary.
