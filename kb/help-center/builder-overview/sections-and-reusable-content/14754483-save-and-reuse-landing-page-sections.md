## Article
If a section performs well, save it once and reuse it across future landing pages using **My templates**.

## Save a section as a template

1. Select a section in the canvas or Layout list.
2. Click **Save section as template** (save icon).
3. Enter a template name.
4. Upload a section preview image.
5. Select **Save**.

## Add the saved template to another page

1. Open **Browse Sections**.
2. Go to **My templates**.
3. Select your saved template to insert it.

## Manage saved templates

- Use clear names so your team can find templates quickly.
- Delete outdated templates from **My templates** when needed.

> Treat saved templates as your conversion library: keep only sections that are current, on-brand, and usable across campaigns.

## Troubleshooting

**I cannot save the template.**

- Template name and preview image are both required.

**I cannot find the template later.**

- Open **Browse Sections** and check the **My templates** category.
