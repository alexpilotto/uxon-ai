## Article
You can edit block-level content inside a section to make targeted updates without changing the full section template.

## Find blocks inside a section

1. Open **Layout** and select your section.
2. Expand the section's block list in the sidebar.
3. Click a block to focus it and edit settings.

## What you can do at block level

- Open block settings (gear icon)
- Hide/show specific blocks
- Edit text, media, links, and style controls

## Recommended block editing workflow

1. Select the section first, then target the block.
2. Make one focused block change.
3. Preview, then continue with the next block.

> Block-level edits are ideal for quick optimization rounds where you only need to adjust one part of a section.

## Troubleshooting

**I cannot find the block I want to edit.**

- Select the section in Layout first, then check its block list.

**A block disappeared from the page.**

- Check block visibility and re-enable it if it was hidden.
