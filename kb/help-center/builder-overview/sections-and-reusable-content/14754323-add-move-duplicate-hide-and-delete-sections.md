## Article
Section management is the fastest way to improve page clarity and conversion flow. Use this guide to control structure without slowing down your build process.

## Where to manage sections

- **Layout panel**: full list of all sections.
- **Canvas overlay**: quick actions on the selected section.

## Add a section

1. Open **Layout**.
2. Select **Add**.
3. Choose a template in **Browse Sections**.

## Move sections

1. In the Layout list, drag a section using the handle on the left.
2. Drop it where you want it in the flow.

This is useful when you want to change narrative order (for example moving proof higher on the page).

## Duplicate a section

- Use the **duplicate** icon on the section row or section overlay.
- Edit the copied section for a faster variation.

## Hide or show a section

- Use the **eye** icon to hide or show a section.
- Hidden sections stay in Builder so you can restore them later.

## Delete a section

- Use the **trash** icon in Layout or on the selected section overlay.
- If you delete by mistake, use **Undo** immediately.

## Extra control tips

- Double-click a section name in Layout to rename it.
- You can also hide specific blocks inside a section when you need lighter edits.

> Best practice: hide first, then delete later. This keeps your team flexible during review rounds.

## Troubleshooting

**I cannot find a section after edits.**

- Check if it was hidden. Use the visibility icon in Layout to show it again.

**Reordering feels slow.**

- Rename sections first so the list is easier to scan while dragging.
