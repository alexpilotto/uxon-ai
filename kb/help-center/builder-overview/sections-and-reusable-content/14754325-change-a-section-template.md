## Article
Use template swap when you want a different section design while keeping your overall landing page structure intact.

## Why change a template

- Test a different visual style for the same section role.
- Adapt layout for readability on mobile.
- Refresh a section without rebuilding the entire page.

## How to change a section template

1. Select the section in canvas or Layout.
2. Open **Section settings** (gear icon).
3. Under **Template**, click the template selector row.
4. In **Change Template**, browse or search and select a new template.
5. Review the section and adjust content/settings as needed.

## What changes after a swap

- The section uses the new template structure and defaults.
- You may need to re-check copy length, media fit, and CTA styling.
- Use **Undo** if you want to revert quickly.

## Best-practice workflow

1. Duplicate the section first if you want a safe fallback.
2. Swap template on the duplicate.
3. Compare in Desktop and Mobile preview before saving.

> Template swaps are ideal for rapid iteration. Use them during optimization sprints to test layout direction faster.

## Troubleshooting

**The new section layout does not match my old content.**

- This is expected after template changes. Revisit section settings and tune content to the new structure.

**I changed the template by mistake.**

- Use **Undo** immediately, or swap back to your previous template.
