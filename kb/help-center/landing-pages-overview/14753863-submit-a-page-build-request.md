## Article
Use **Page Builds** to request a landing page build from the UXON team. You complete a structured brief, submit payment when required, and track delivery status from one place.

## Before you start

To submit a request, make sure your user role can access Page Builds. Member users cannot create requests.

Have this information ready:

- Whether this is for a **new** or **existing** client account
- A clear project name
- Language, funnel type, optimization event, buyer group, and target market
- Offer details and ideal customer profile (ICP)
- Asset links (for example Figma, Drive, brand files, copy docs)

## How to submit a request

1. Open **Dashboard** and go to **Page Builds**.
2. Select **Create Page Build**.
3. Complete the request flow from step 1 to step 11.
4. Add at least one **Asset link** in the final step.
5. Select **Proceed to Checkout** (or **Submit Request** when no payment is required).

## What the brief asks for

The form is designed to capture everything needed to build a high-quality landing page quickly. You will provide:

- Client details (new client brand + website, or existing client account)
- Build request name
- Language and funnel type (Lead Generation or Ecommerce)
- Primary optimization event (Lead, Purchase, Free trial start, Book call, Demo booked, or custom)
- Buyer group (B2B or B2C) and market
- Offer details, ICP details, and optional comments

> If you enter a website without protocol, UXON normalizes it to a secure **https://** URL automatically.

## What happens after submission

- Your request appears in the Page Builds table with a unique request ID.
- You receive a confirmation email with request details and payment status.
- You can open the request at any time to review brief details and current status.

## Troubleshooting

**I cannot create a request.**

- Check your role. Member users cannot create build requests.
- Confirm the Page Builds feature is enabled in your plan.

**I cannot continue to the next step.**

- Complete all required fields in the current step.
- For custom optimization event or market, the custom text field is required.
